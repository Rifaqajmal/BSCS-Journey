# Computer Organization & Assembly Language (COAL)

This folder contains laboratory work and assembly language programs completed during the Computer Organization & Assembly Language (COAL) course in Semester 3 of the BS Computer Science program at UET Mardan.

## Topics Covered

* Basic Assembly Program Structure
* Input and Output Operations
* Data Types and Memory Management
* Loops and Conditional Statements
* Arrays and String Manipulation
* Addressing Modes
* Stack Operations
* Procedures and Macros
* Multiplication and Division Instructions
* Shift and Rotate Operations
* Control Flow and Branching

## Files Included

* COAL Laboratory Report
* Assembly Language Source Codes (.asm)
* Lab Exercises and Practical Implementations

## Technologies Used

* Assembly Language
* EMU8086 / MASM Concepts

## Learning Outcomes

* Understanding computer architecture fundamentals
* Working with CPU registers and memory
* Low-level programming concepts
* Instruction execution and addressing mechanisms
* Assembly language problem solving
