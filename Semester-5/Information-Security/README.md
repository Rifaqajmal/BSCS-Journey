# Information Security

This folder contains assignments and coursework completed during the Information Security course in Semester 5 of the BS Computer Science program at UET Mardan.

## Topics Covered

### Security Fundamentals

* Information Security Concepts
* Confidentiality, Integrity & Availability (CIA Triad)
* Security Threats & Vulnerabilities

### Cryptography

* Encryption Techniques
* Symmetric & Asymmetric Encryption
* Hashing Concepts

### Network Security

* Secure Communication
* Authentication Mechanisms
* Access Control

### Cyber Security Concepts

* Malware & Attacks
* Security Policies
* Risk Management

## Files Included

* Course Assignments
* Security Reports
* Research Activities

## Learning Outcomes

* Understanding information security principles

* Basic cryptographic concepts
