# Design & Analysis of Algorithms (DAA)

This folder contains algorithm implementations and programming exercises completed during the Design & Analysis of Algorithms course in Semester 4.

* Insertion Sort
* Bubble Sort
* Selection Sort
* Merge Sort
* Quick Sort
* Counting Sort
* Heap Sort

### Searching Algorithms

* Binary Search

### Trees

* Binary Search Tree (BST)

### Dynamic Programming

* Fibonacci Implementation

### Greedy Algorithms

* Fractional Knapsack

### Graph Algorithms

* Dijkstra's Algorithm
* Kruskal's Algorithm
* Topological Sorting

## Technologies Used

* C++

* Algorithm design principles
* Time and space complexity analysis
* Recursive problem solving
* Graph algorithm implementation
* Dynamic programming techniques
* Greedy strategy applications
