# 📱 Mobile Application Development (MAD)

This folder contains all laboratory work, assignments, and projects completed during the Mobile Application Development course in Semester 6 of BS Computer Science at UET Mardan.

The course focused on modern Android application development using Flutter and Dart, covering user interface design, Firebase integration, authentication systems, payment gateways, local storage, biometric authentication, Google Maps, and mobile deployment.
